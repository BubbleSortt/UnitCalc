Cтоимость привлечения клиента CAC (customer acquisition cost)
Cтоимость привлечения клиента — это совокупные затраты, требующиеся для конвертации потенциального клиента в фактического покупателя. CAC = Все затраты на привлечение нового клиента / Количество новых клиентов
Cтоимость клика CPC (Cost Per Click)
CPC показывает, сколько вы платите за клик по вашему объявлению CPC = Рекламный бюджет / Количество кликов или указываем вручную
